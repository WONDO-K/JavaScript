# exchange_currency 💱

JavaScript로 구현한 간단한 환율 변환 시스템입니다.  
사용자가 선택한 두 통화 간의 환율에 따라 금액을 변환할 수 있도록 설계된 프론트엔드 연습 프로젝트입니다.

## 📌 주요 기능

- 주요 통화 간 환율 변환 (예: USD, EUR, KRW 등)
- 사용자가 입력한 금액을 기준으로 자동 변환
- 간단한 셀렉트 박스를 통한 통화 선택 UI
- 실시간 환율 API 또는 고정 환율 기반 처리 (옵션)

## 🛠️ 사용 기술

- JavaScript
